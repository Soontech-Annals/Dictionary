# Item memory
(noun)

## 1. A device that uses codes to store and retrieve itemss addressably

| IO  | Type | Description                     |
| --- | ---- | ------------------------------- |
| In  | Code | Code                            |
| In  | 0-1  | Store/retrieve mode selector    |
| In  | Item | Item to store in store mode     |
| Out | Item | Retrieved item in retrieve mode |
